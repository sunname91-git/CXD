Поле для ввода пароля пользователя
* •	макс 128 символов
* •	тип данных: строка (string)
* •	латиница (A-Z, a-z), цифры (0-9), спецсимволы: !"№;%:?*()_+@#$^&-={}[]<>/.,
* •	скрытие символов при вводе
* •	возможность отображения пароля по запросу пользователя (например, через иконку "показать пароль")
* •	поддержка UTF-8
* •	требования:
> - Минимальная длина пароля — 8 символов;
> - Содержит хотя бы одну заглавную букву;
> - Содержит хотя бы одну строчную букву;
> - Содержит хотя бы одну цифру;
> - Содержит хотя бы один специальный символ.
