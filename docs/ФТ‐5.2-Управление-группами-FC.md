## Описание
Система хранения данных (СХД) должна поддерживать ручное управление группами Fibre Channel (FC), включая создание, редактирование, удаление групп и добавление в них инициаторов (хостов).
Группа Fibre Channel — это логическая группа инициаторов (хостов), объединённых по признаку принадлежности к одному или нескольким LUN, с целью централизованного управления доступом к ресурсам хранения.
Управление группами FC должно быть доступно через WEB-интерфейс (GUI) и командную строку (CLI).
## Требуемый функционал
1. Создание группы FC
* •	Администратор может создать новую группу FC, указав уникальное [имя](https://github.com/sunname91-git/API/wiki/Rule_07_%D0%9D%D0%B0%D0%B8%D0%BC%D0%B5%D0%BD%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5-%D1%81%D1%83%D1%89%D0%BD%D0%BE%D1%81%D1%82%D0%B5%D0%B9-(Name)) группы.
* •	Группа изначально не содержит инициаторов.
2. Ручное добавление инициаторов (WWPN)
* •	Администратор может вручную добавить один или несколько инициаторов в группу FC, указав их [WWPN](https://github.com/sunname91-git/API/wiki/Rule_02_WWN).
* •	Система должна проверять корректность формата WWPN перед добавлением.
* •	При попытке добавить уже существующий WWPN в ту же группу система выводит информационное сообщение о дублировании.
3. Редактирование группы FC
* •	Администратор может изменять имя группы.
* •	Администратор может добавлять или удалять инициаторы (WWPN) из группы.
* •	При попытке удалить активный WWPN (с установленной сессией и/или активным I/O) система блокирует операцию и выводит информативное предупреждение.
4. Удаление группы FC:
* •	Администратор может удалить группу FC.
* •	Перед удалением система проверяет, не назначена ли группа к LUN (логическому устройству хранения).
* •	Если группа используется (привязана к LUN), удаление запрещается до отвязки.
5. Просмотр и мониторинг:
* •	Система должна отображать:
> - Список всех групп FC.
> - Состояние каждой группы (активен/неактивен  — при наличии соответствующей информации от FC-адаптера).
> - Список инициаторов, ассоциированных с группой.
* •	Все действия по управлению группами FC должны фиксироваться в [журнале действий пользователя](https://github.com/sunname91-git/API/wiki/_new#scroll-bookmark-87).
